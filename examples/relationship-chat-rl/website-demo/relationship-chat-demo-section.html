Relationship Chat RL Demo

Watch a small model learn how to talk to your partner without stepping on landmines

Scenario: “You forgot our anniversary again”

We turned the classic “how to chat with your girlfriend/wife” problem into a reinforcement-learning task. The model generates a reply; a hidden rubric scores it on empathy and penalizes dismissive phrases like “what's the big deal” or “calm down”. PPO on a CPU-only laptop pushes the verify success rate up over successive iterations.

  • 🧠 Base model: Qwen/Qwen3.5-0.8B + LoRA
  • 🏛️ Infra: Arena rollout / verify / trajectory plane
  • 📈 Dashboard: live verify-success-rate trend
  • 🦙 Deploy: import the LoRA adapter directly into Ollama
Run the demo 
git clone https://github.com/albert-lv/OpenAgora.git
cd OpenAgora/examples/relationship-chat-rl
docker compose up --build

Open http://localhost:9091 and watch the Verify Stats tab.
